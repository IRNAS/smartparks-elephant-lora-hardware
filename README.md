# SmartParks Elephant LoRa addon board

Latest Version V1

Board has Murata module that containtes STM32L072 ARM Cortex M0+ 32 bit 
microcontroller and Semtech SX1276, and also a EByte Lora module with 
new Semtech chip SX1262.
